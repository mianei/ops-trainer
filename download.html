GitHub 开源 · 3 步搞定

  1. 打开 仓库 Settings，确认是 Public（公开），别人才能看到和下载
  2. 代码已在仓库里；别人点绿色 Code → Download ZIP 就能下载
  3. （可选）在 Releases 发一个版本号如 v4.34，更像正式「发版」
