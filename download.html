说明

CVassistant 以 Web 应用为主：注册登录后使用，简历与练习记录保存在你的账号下。桌面版为 Electron 封装，内容与线上一致。

  • 在线版：需邀请码注册；部署时设 AUTH_DISABLED=0 并配置 Upstash 与 REGISTER_INVITE_CODE
  • 源码下载：仓库页 Code → Download ZIP，或见 README 部署说明
  • 桌面版：在 Releases 查找 Windows-portable 附件
