## AI 产品项目教练（skill 融合）

Skill 源码位于 `skills/ai-product-project-coach/`（Cursor 同步在 `.cursor/skills/`），包含：

- `references/intent-routing.md` — 意图路由（完整方案 / 项目方向 / 迭代 / 简历诊断 / AI PM 答疑 / 面试防御）
- `references/assessment-framework.md`、`harness-project-logic.md`、`project-patterns.md` 等

Web 端「**项目教练**」Tab 或训练智能体输入「诊断 AI 产品简历」等关键词，会调用 `/api/coach`，按模式加载对应 reference 并生成结构化辅导。

```bash
# 更新 skill：从 coach 仓库复制
Copy-Item -Recurse -Force ../coach/ai-product-project-coach skills/ai-product-project-coach
```

## Coach 面经导入

```bash
node scripts/import-coach-docx.js
```

从 `Desktop/coach/面经_去重版.docx` 导入 **218 道** AI PM 题 + **精选 10 题话术** → `interview-bank-ai-pm.json` / `interview-bank-ai-pm-curated.json` / 更新 `interview-rag-feed.json`。

## 构建题库

```bash
node scripts/build-interview-bank-v2.js
```

## 跑评分一致性评测

```bash
DEEPSEEK_API_KEY=sk-xxx node scripts/eval-scoring.js
```

## 环境变量

| 变量 | 说明 |
|------|------|
| `DEEPSEEK_API_KEY` | 必填 |
| `OPENAI_API_KEY` | 语音转写（可选） |
| `PM_REVIEW_MODE` | 默认 1，产品岗五维点评 |

## 待你手动完成（规划第 1–6 天）

1. 从牛客/小红书/知乎补充带原链接的真题到 Excel，再导入 feed 或 bank
2. 找 1–2 同学对 `eval/scoring-golden.json` 做人工复核标注
3. 跑 eval-scoring 记录 Spearman，迭代 badcase prompt
